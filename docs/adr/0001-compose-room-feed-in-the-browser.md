# Compose the Room Feed in the Browser

The product should behave like a simple hosted live studio, not a source-routing tool. For v1, the browser app composes everyone currently in the Room into one Composed Room Feed before it is sent onward to the broadcast destination, so OBS or another bridge is not responsible for arranging individual participant sources. The first milestone must validate this composed output, not only prove that Host and Guests can talk to each other.
